# Self-Hosted Network Lab

This personal lab simulates a networked service environment, providing hands-on experience in Linux server administration, real-time monitoring, and incident response.  

## Skills Demonstrated
- Managing Linux users and enforcing access control
- Configuring firewalls and secure tunneling
- Automating service monitoring and optimizing resource usage
- Analyzing logs and monitoring server performance
- Responding to issues and troubleshooting in real time
